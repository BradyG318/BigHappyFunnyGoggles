# Raspberry Pi 5 / ARM64-friendly requirements for BigHappyFunnyGoggles
# Goal: maximize the chance that `pip install -r requirements_pi5.txt`
# succeeds on a Pi 5 by avoiding Windows-only packages, version pin conflicts,
# and unnecessary hard pins that force source builds on ARM.
#
# Recommended environment:
#   - Raspberry Pi OS 64-bit or Ubuntu 64-bit
#   - Python 3.11
#   - Latest pip/setuptools/wheel before install
#
# Important:
# - MediaPipe is intentionally NOT included here. Google states that MediaPipe
#   PyPI does not provide aarch64 Python wheel files, so plain pip install on
#   Raspberry Pi is not reliable.
# - tensorflow-intel is intentionally removed because it is for Intel/Windows,
#   not Raspberry Pi ARM.
# - Most strict pins were removed so pip can choose ARM-compatible wheels.

setuptools
wheel
